# Automatic-File-Sorter
A python project that helps to automatically organizes files into folders based on their file extensions.

## Features
- Creates folders automatically if they don't exist.
- Moves CSV files into the "csv files" folder.
- Moves JPG images into the "image files" folder.
- Moves TXT files into the "text files" folder.
- Moves PDF files into the "pdf files" folder.
- Moves DOCX files into the "docx files" folder.
- Moves MP4 videos into the "vid files" folder.

## Technologies Used
- Python
- os module
- shutil module

## How to Use
1. Open the notebook.
2. Change the path variable to the folder you want to organize.
3. Run all cells.
4. The files will be sorted automatically.
